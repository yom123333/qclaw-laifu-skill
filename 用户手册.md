## 一、安装（5分钟）

### 1.1 安装 Python 依赖
```bash
pip install pandas numpy requests pyyaml openai
```

### 1.2 复制引擎文件
将本目录下的 `engine/` 文件夹复制到：
```
C:\Users\你的用户名\.qclaw\engine\
```

将 `config.yaml` 复制到：
```
C:\Users\你的用户名\.qclaw\config.yaml
```

### 1.3 配置（可选）
编辑 `C:\Users\你的用户名\.qclaw\config.yaml`：
```yaml
deepseek:
  api_key: "sk-你的key"    # 复盘AI摘要用，不填也能跑

tdx:
  vipdoc_path: "C:\\zd_zsone\\T0002\\vipdoc"  # 通达信路径，不填用演示模式
```

### 1.4 创建 QClaw 智能体
1. 打开 QClaw → 智能体管理 → 新建智能体
2. 名称：**来福**
3. 模型：**DeepSeek**
4. 描述栏：粘贴 `来福智能体提示词.txt` 全部内容
5. 保存

### 1.5 安装策略知识库（可选但推荐）
将 `skills/laifu/` 文件夹复制到：
```
C:\Users\你的用户名\.qclaw\skills\laifu\
```

## 五、日常使用流程

### 交易日流程
- **08:55** — 自动弹窗盘前简报（隔夜外盘+今日日历风险）
- **09:25** — 自动弹窗开盘快照
- **盘中** — 风险哨兵和持仓监控自动静默运行，有告警才弹窗
- **14:45** — 自动扫描今日信号，弹窗简报
- **15:30** — 自动生成策略日记
- **20:00** — 自动生成完整复盘

### 主动操作
- **盘前** — 问来福"今天风险怎么样"
- **盘中** — 问来福"持仓面板"看浮盈浮亏
- **盘后** — 问来福"分析最近胜率"
- **周末** — 问来福"跑一下回测"调参数

## 六、引擎说明

### 回测引擎 (backtest.py)
```bash
--strategy longhui        # 策略选择: longhui / attack
--fast                    # 极速测试(10只股票)
--montecarlo 500          # 蒙特卡洛模拟500次
--scan shrink_ratio --from 0.25 --to 0.55 --step 0.05  # 参数扫描
--compare                 # 双策略对比
```

### 信号扫描 (scanner.py)
```bash
--scan                    # 全市场扫描（需vipdoc）
--scan --quick            # 快速演示模式
--check 代码              # 单股检查
--history 30              # 查看30天历史信号
```

### 持仓监控 (monitor.py)
```bash
--activate                # 将信号入库为持仓
--check                   # 检查出场条件（有触发才输出）
--dashboard               # 持仓面板
--alerts                  # 今日预警记录
```

### 风险哨兵 (riskmon.py)
```bash
--check                   # 运行一次风险检查（静默）
--premarket               # 盘前简报
--calendar                # 日历风险
```

### 复盘/日记 (review.py)
```bash
--today                   # 今日复盘
--today --personal        # 策略日记
--date 2026-05-24         # 指定日期
```

### AI导师 (coach.py)
```bash
--winrate                 # 胜率分析
--drawdown                # 回撤诊断
--optimize                # 参数优化建议
--full                    # 完整诊断
```

## 七、常见问题

**Q: 没有通达信vipdoc能用吗？**
A: 能。回测用 `--fast`、扫描用 `--quick`，自动使用演示数据。配了通达信后跑真实数据。

**Q: 需要配DeepSeek key吗？**
A: 不必须。策略规则、信号扫描、持仓监控、风险检查都不需要。只有复盘AI摘要需要。

**Q: 定时任务会刷屏吗？**
A: 不会。风险检查和持仓监控是静默模式，正常时不输出。只有触发告警才弹窗。

**Q: 怎么更新？**
A: 重新下载最新 engine/ 文件覆盖即可。`config.yaml` 和 `positions.json` 不会被覆盖。
